## 2. 安装与启动

### 系统要求

- **操作系统**：Windows 10 / 11 64 位
- **内存**：建议 4GB 以上
- **硬盘空间**：至少 100MB

### 安装步骤

1. **下载安装包**
   - 从 [GitHub Releases](https://github.com/ruizznav/MengBao_Clipboard_Setup/releases/latest) 下载
   - `萌宝剪贴板 v5_Setup.exe`（安装版）
   - 或 `萌宝剪贴板 v5.exe`（便携版，无需安装）

2. **运行安装程序**
   - 双击 `萌宝剪贴板 v5_Setup.exe`
   - 按照安装向导提示完成安装
   - 建议保持默认安装路径

3. **启动软件**
   - 安装完成后，萌宝剪贴板会自动启动
   - 软件启动后，图标会出现在**系统托盘**（右下角）
   - 首次运行会显示主窗口

### 首次运行设置

1. **系统托盘图标**
   - 萌宝图标会显示在系统托盘（右下角）
   - 如果看不到，可能需要点击「向上箭头」展开隐藏图标

2. **全局唤醒快捷键**
   - 默认：`Ctrl + Shift + V`
   - 可以在「偏好设置」→「快捷键」中修改

3. **开机自启（可选）**
   - 打开「偏好设置」→「通用设置」
   - 勾选「登录时启动」

## 4. 剪贴板功能

### 4.1 自动记录

#### 原理
萌宝剪贴板会监听系统剪贴板，当你复制内容时，自动记录到当前主分类。

#### 触发条件
- **复制文本**：`Ctrl + C` 或右键复制
- **复制图片**：截图、拍照等
- **复制文件**：文件管理器中的文件复制

#### 记录位置
- 自动添加到**当前选中的主分类**的剪贴板列表
- 如果当前在主分类的「全部」子分类，则记录所有类型
- 如果在特定子分类（如「文本」），仍会记录所有类型，但列表显示会按筛选条件过滤

### 4.2 快速搜索

#### 打开搜索
- **方法一**：点击搜索框
- **方法二**：按 `Ctrl + F`
- **方法三**：按 `F` 键（需先聚焦到搜索框）

#### 搜索技巧
- **实时搜索**：输入关键词后，列表实时显示匹配结果
- **清除搜索**：删除搜索框内容或按 `Esc`
- **搜索范围**：搜索会匹配文本内容、图片名称、文件路径

### 4.3 粘贴操作

#### 双击粘贴
- **操作**：双击剪贴板列表中的任意条目
- **效果**：
  1. 将该条目内容复制到系统剪贴板
  2. 自动粘贴到当前活跃窗口
- **适用场景**：
  - 粘贴文本到文档
  - 粘贴图片到聊天窗口
  - 粘贴文件路径到资源管理器

#### 单击选中 + Enter 粘贴
- **操作**：
  1. 单击选中条目
  2. 按 `Enter` 键
- **效果**：同双击粘贴

#### 右键菜单粘贴
- **操作**：
  1. 右键点击条目
  2. 选择「粘贴」
- **效果**：同双击粘贴

### 4.4 右键菜单详解

#### 打开右键菜单
- **方法一**：右键点击剪贴板列表中的条目
- **方法二**：选中条目后按 `Shift + F10`

#### 菜单选项详解

| 选项 | 功能 | 快捷键 |
|---|---|---|
| 粘贴 | 粘贴选中条目到当前窗口 | `Enter` |
| 复制 | 复制选中条目到剪贴板 | - |
| 删除 | 删除选中条目 | `Delete` |
| 上移 | 上移选中条目 | `Ctrl + ↑` |
| 下移 | 下移选中条目 | `Ctrl + ↓` |
| 备注 | 为选中条目添加备注 | - |
| 移动到 | 将条目移动到其他分类 | - |
| 粘贴为纯文本 | 粘贴时仅保留纯文本 | - |
| 粘贴为路径 | 粘贴文件路径 | - |
| 预览图片 | 预览图片内容 | - |
| 下载图片 | 下载网络图片到本地 | - |
| 在文件资源管理器中显示 | 打开文件所在位置 | - |
| 在浏览器访问 | 在浏览器中打开链接 | - |
| 发送邮件 | 将内容作为邮件发送 | - |
| 导出为文件 | 将内容导出为文件 | - |

#### 移动到分类
1. 右键点击条目
2. 选择「移动到」
3. 选择目标分类
4. 条目会移动到目标分类

### 4.5 自由排序

#### 拖拽排序
1. 鼠标按住条目
2. 拖动到目标位置
3. 释放鼠标

#### 上移/下移
- **上移**：选中条目后，按 `Ctrl + ↑`
- **下移**：选中条目后，按 `Ctrl + ↓`

### 4.6 自动排序

#### 功能说明
当「自动排序」开启时，复制已存在的内容会自动排列到最前面。

#### 开启/关闭
1. 打开「偏好设置」→「剪贴板」→「内容设置」
2. 勾选/取消勾选「自动排序」

### 4.7 自动粘贴

#### 功能说明
当「自动粘贴」开启时，鼠标左键操作（单击/双击）会快速粘贴内容至输入位置。

#### 开启/关闭
1. 打开「偏好设置」→「剪贴板」→「内容设置」
2. 勾选/取消勾选「自动粘贴」
3. 选择触发方式：「单击」或「双击」

## 5. 分类管理

### 5.1 双层分类系统

萌宝剪贴板采用**双层分类系统**，帮助你高效整理剪贴板内容。

#### 第一层：主分类
- **作用**：按用途或项目分类
- **默认分类**：`默认`
- **自定义分类**：点击主分类栏的 `+` 添加

**示例**：
- `默认`：未分类的内容
- `工作`：工作相关的复制内容
- `学习`：学习相关的复制内容
- `生活`：生活相关的复制内容

#### 第二层：子分类
- **作用**：按内容类型筛选
- **选项**：
  - `全部`：显示该主分类下的所有内容
  - `文本`：仅显示文本内容
  - `图片`：仅显示图片内容
  - `文件`：仅显示文件内容

**使用示例**：
> 在「工作」→「图片」中快速找到工作截图  
> 在「学习」→「文本」中翻阅昨天的笔记

### 5.2 管理主分类

#### 添加主分类
1. 点击主分类栏的 `+`
2. 输入分类名称
3. 按 `Enter` 确认

#### 重命名主分类
1. 右键点击要重命名的分类
2. 选择「重命名」
3. 输入新名称
4. 按 `Enter` 确认

#### 删除主分类
1. 右键点击要删除的主分类
2. 选择「删除」
3. 确认删除

> **注意**：删除主分类不会删除其中的内容，内容会移动到「默认」分类。

### 5.3 切换子分类

#### 操作方法
- **方法一**：点击子分类栏中的选项
- **方法二**：使用快捷键切换（需在「快捷键」中设置）

#### 子分类说明
- `全部`：显示该主分类下的所有内容
- `文本`：仅显示文本内容（纯文本、RTF、HTML）
- `图片`：仅显示图片内容
- `文件`：仅显示文件内容（文件路径）

#### 最大保留条数
- **功能**：设置保存的最大条目数
- **设置方法**：
  1. 打开「偏好设置」→「历史记录」
  2. 调整「最大保留条数」滑块
  3. 或输入具体数字
- **建议**：
  - 内存充足：1000+

#### 保留时长
- **功能**：设置条目保留时长
- **设置方法**：
  1. 打开「偏好设置」→「历史记录」
  2. 输入保留天数

#### 时间范围
- **功能**：设置删除操作的时间范围
- **选项**：
  - `时间不限`
  - `过去 1 小时`
  - `过去 24 小时`
  - `过去 7 天`
  - `过去 30 天`
  - `自定义`
- **设置方法**：
  1. 打开「偏好设置」→「历史记录」
  2. 选择「时间范围」

- **操作方法**：
  1. 打开「偏好设置」→「历史记录」
  2. 点击「删除历史记录」
  3. 确认删除
- **注意**：清理后无法恢复，建议先备份。

## 10. 常见问题

### 10.1 安装与启动

#### Q：安装后找不到软件图标？
A：萌宝剪贴板启动后，图标会显示在**系统托盘**（右下角）。如果看不到，可能需要点击「向上箭头」展开隐藏图标。

#### Q：如何完全退出萌宝剪贴板？
A：右键系统托盘图标 →「退出」。

#### Q：如何暂时关闭剪贴板监听？
A：右键系统托盘图标 →「停止监听」。

#### Q：如何重新开启剪贴板监听？
A：右键系统托盘图标 →「开启监听」。

### 10.2 功能使用

#### Q：为什么复制的内容没有记录？
A：可能原因：
1. 剪贴板监听已关闭 → 右键系统托盘图标 →「开启监听」
2. 达到了最大记录数 → 打开「偏好设置」→「历史记录」，增加最大保留条数
3. 复制的内容不支持 → 萌宝剪贴板支持文本、图片、文件
4. 历史已有复制 → 「偏好设置」 → 「自动排序」

#### Q：为什么浏览器网址会复制两条记录？
A：可能原因：
1. 当你从网页复制时，浏览器（Chrome/Edge/Edge 等）并不是一次性把所有格式写入剪贴板的，而是分两次写入第一次：写入 HTML 格式 → 触发 onClipboardChange，result.html 有值
极短时间后：写入纯文本格式 → 再次触发 onClipboardChange，result.text 也有值。 → 「偏好设置」 → 「复制为纯文本」
   
#### Q：如何修改全局唤醒快捷键？
A：打开「偏好设置」→「快捷键」→ 修改「打开剪贴板窗口」。

#### Q：如何删除单条记录？
A：选中条目后按 `Delete`，或右键点击条目 →「删除」。

#### Q：如何批量删除记录？
A：打开「偏好设置」→「历史记录」→「删除历史记录」清空所有记录。

#### Q：如何添加备注？
A：右键点击条目 →「备注」→ 输入备注内容 → 确认。

### 10.3 分类管理

#### Q：如何创建新分类？
A：点击主分类栏的 `+` → 输入分类名称 → 按 `Enter` 确认。

#### Q：如何删除分类？
A：右键点击要删除的分类 →「删除」→ 确认删除。

> **注意**：删除分类不会删除其中的内容，内容会移动到「默认」分类。

#### Q：如何重命名分类？
A：右键点击要重命名的分类 →「重命名」→ 输入新名称 → 按 `Enter` 确认。

### 10.4 备份与恢复

#### Q：备份文件保存在哪里？
A：默认保存在下载目录，你可以在备份时选择其他位置。

#### Q：如何定期自动备份？
A：打开「偏好设置」→「数据备份」→「备份和恢复」→ 勾选「自动备份」→ 设置备份频率和位置。

#### Q：恢复数据会覆盖当前内容吗？
A：是的，恢复数据会覆盖当前所有内容。建议先备份当前数据。

#### Q：备份文件可以迁移到 other 电脑吗？
A：可以，将备份文件（`.mbak`）复制到 other 电脑，然后在萌宝剪贴板中恢复即可。

### 10.5 其他问题

#### Q：萌宝剪贴板会占用很多内存吗？
A：不会，萌宝剪贴板基于 Tauri v2 + Rust 构建，内存占用非常低（通常 < 50MB）。

#### Q：支持多语言吗？
A：目前支持：
- 简体中文（zh-CN）
- English (en-US)
- 日本語 (ja-JP)
- 繁體中文 (zh-TW)

#### Q：如何更新到最新版本？
A：开启「自动检查更新」后，萌宝剪贴板会自动提示更新。或关注 [GitHub Releases](https://github.com/ruizznav/MengBao_Clipboard_Setup/releases/latest)，下载最新版本覆盖安装即可。

#### Q：萌宝剪贴板是开源的吗？
A：是的，本项目基于 [Apache 2.0](LICENSE) 协议发布，基于 [EcoPaste](https://github.com/ayangweb/EcoPaste) 修改。
